## Endpoint

```
GET /list
```

Get a complete list of all sites currently in the webring. This endpoint does not require any authentication or query parameters.

## Query Parameters

This endpoint does not accept any query parameters.

## Response

<ResponseField name="sites" type="array" required>
  Array of all sites in the webring.
  
  <Expandable title="Site Object">
    <ResponseField name="url" type="string" required>
      Full URL of the site.
    </ResponseField>
    
    <ResponseField name="name" type="string" required>
      Display name of the site.
    </ResponseField>
    
    <ResponseField name="image" type="string">
      Banner or avatar image URL.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Short description of the site.
    </ResponseField>
  </Expandable>
</ResponseField>

## Status Codes

- **200 OK** - Successfully returned the list of sites

## Examples

<CodeGroup>

```bash curl
curl "https://openring.dev/list"
```

```javascript JavaScript
const response = await fetch('https://openring.dev/list');
const data = await response.json();
console.log(data.sites);
```

```python Python
import requests

response = requests.get('https://openring.dev/list')
data = response.json()
print(data['sites'])
```

</CodeGroup>
