## DELETE /admin/sites

Removes an existing site from the webring. This endpoint requires admin authentication via Bearer token.

<Warning>
  This endpoint requires authentication. You must include a valid Bearer token in the Authorization header.
</Warning>

### Authentication

This endpoint requires a Bearer token in the Authorization header:

```
Authorization: Bearer YOUR_ADMIN_TOKEN
```

### Request Body

<ParamField body="url" type="string" required>
  Full URL of the site to remove
  
  Example: `https://example.com`
</ParamField>

### Response

<Expandable title="200 - Site successfully removed">
  Returns the removed site object.
  
  <ResponseField name="url" type="string" required>
    Full URL of the site
  </ResponseField>
  
  <ResponseField name="name" type="string" required>
    Display name of the site
  </ResponseField>
  
  <ResponseField name="image" type="string" optional>
    Banner or avatar image URL
  </ResponseField>
  
  <ResponseField name="description" type="string" optional>
    Short description of the site
  </ResponseField>
  
  ```json
  {
    "url": "https://example.com",
    "name": "Example Site",
    "image": "https://example.com/banner.png",
    "description": "A cool website"
  }
  ```
</Expandable>

<Expandable title="400 - Invalid request body">
  The request body is invalid or missing the required URL field.
  
  <ResponseField name="error" type="string">
    Error message describing what went wrong
  </ResponseField>
  
  ```json
  {
    "error": "Invalid request body"
  }
  ```
</Expandable>

<Expandable title="401 - Unauthorized">
  Invalid or missing admin token in the Authorization header.
  
  <ResponseField name="error" type="string">
    Error message describing the authentication failure
  </ResponseField>
  
  ```json
  {
    "error": "Unauthorized. Invalid admin token"
  }
  ```
</Expandable>

<Expandable title="404 - Site not found">
  The specified site was not found in the webring.
  
  <ResponseField name="error" type="string">
    Error message indicating the site was not found
  </ResponseField>
  
  ```json
  {
    "error": "Site not found in webring"
  }
  ```
</Expandable>

<Expandable title="500 - Failed to remove site">
  An internal server error occurred while removing the site.
  
  <ResponseField name="error" type="string">
    Error message describing the failure
  </ResponseField>
  
  ```json
  {
    "error": "Failed to remove site"
  }
  ```
</Expandable>

### Example Request

<CodeGroup>

```bash cURL
curl -X DELETE https://api.openring.dev/admin/sites \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.openring.dev/admin/sites', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_ADMIN_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    url: 'https://example.com'
  })
});

const data = await response.json();
```

```python Python
import requests

url = 'https://api.openring.dev/admin/sites'
headers = {
    'Authorization': 'Bearer YOUR_ADMIN_TOKEN',
    'Content-Type': 'application/json'
}
data = {
    'url': 'https://example.com'
}

response = requests.delete(url, headers=headers, json=data)
result = response.json()
```

</CodeGroup>
